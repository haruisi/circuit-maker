回路図メーカー v3.5

v3.4の修正版です。

修正:
- 格子が表示されない問題を修正
  - CSS背景で格子を表示するように変更
- iPad Safariでツールを触れない問題を修正
  - ボタン操作を click ではなく pointerdown で処理
  - ボタン操作がキャンバス操作に吸われないように調整

機能:
- 左側: 部品ツール
- 下側: 導線 / 消す / 全消し / 保存 / コピー
- 消しゴムは、なぞった下にあるものをまとめて削除
- 抵抗は四角い箱型
- 部品は斜め配置OK
- 透明背景PNG保存
- PNGコピー対応（環境により失敗する場合あり）

操作:
- 導線: 下の「導線」を押して格子上をなぞる
- 部品: 左の部品を選び、始点から終点へドラッグ
- 消す: 下の「消す」を押して、消したい範囲をなぞる
- 二本指ダブルタップ: 戻る
- 三本指ダブルタップ: 進む

注意:
- コピーが動かない場合は保存を使ってください。
