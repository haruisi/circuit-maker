回路図メーカー v3.4

変更点:
- よく使う「導線」「消す」「全消し」「保存」「コピー」を下バーに配置
- 消しゴムは、なぞった下にあるものをまとめて削除
- PNGコピー機能を追加
- 抵抗は四角い箱型
- 部品は斜め配置OK
- スイッチは削除済み。接続点2つで代用してください。

使い方:
1. index.html をブラウザで開く
2. 左のツールで部品を選ぶ
3. 導線は下の「導線」を押して、格子上をなぞる
4. 抵抗・電源・コンデンサー・計器などは、始点から終点へドラッグして置く
5. 斜めの格子点へドラッグすると部品も斜めに置ける
6. 下の「保存」で透明背景PNGを保存
7. 下の「コピー」でPNGをクリップボードへコピー

操作:
- 二本指ダブルタップ: 戻る
- 三本指ダブルタップ: 進む

注意:
- PNGコピーは、ブラウザやiOSの制限により動かない場合があります。
- コピーできない場合は「保存」を使ってください。
- GitHub PagesなどのHTTPS環境では、コピーが動きやすいです。
