回路図メーカー v3.6

v3.5からの変更:
- ツールを素早く切り替えたときにも反応しやすいように修正
  - touchstart / pointerdown / mousedown を併用
  - iPad Safariでのイベント重複は内部で防止
- 全消しの確認ダイアログを削除
  - 押したら即全消し
  - ミスした場合は「戻る」または二本指ダブルタップで復元できます

基本機能:
- 左側: 部品ツール
- 下側: 導線 / 消す / 全消し / 保存 / コピー
- 消しゴムは、なぞった下にあるものをまとめて削除
- 抵抗は四角い箱型
- 部品は斜め配置OK
- 透明背景PNG保存
- PNGコピー対応（環境により失敗する場合あり）

操作:
- 導線: 下の「導線」を押して格子上をなぞる
- 部品: 左の部品を選び、始点から終点へドラッグ
- 消す: 下の「消す」を押して、消したい範囲をなぞる
- 全消し: 確認なしで即実行
- 二本指ダブルタップ: 戻る
- 三本指ダブルタップ: 進む

注意:
- コピーが動かない場合は保存を使ってください。
