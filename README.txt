回路図メーカー v3.7

v3.6からの変更:
- 画像コピー機能を削除
  - ユーザー環境で動かなかったため、保存に絞りました
- 左ツールバーを2列固定に変更
  - スクロール式を廃止
  - スクロールしようとしてツールを誤タップする問題を避けます
- ツール切り替えをpointerdownで即反応に変更
  - 余計なtouchstart / mousedown併用をやめ、挙動を安定させました
- 全消しは確認なしで即実行
  - 戻る操作で復元できます

基本機能:
- 左側: 部品ツール2列固定
- 下側: 導線 / 消す / 全消し / 保存
- 消しゴムは、なぞった下にあるものをまとめて削除
- 抵抗は四角い箱型
- 部品は斜め配置OK
- 透明背景PNG保存

操作:
- 導線: 下の「導線」を押して格子上をなぞる
- 部品: 左の部品を選び、始点から終点へドラッグ
- 消す: 下の「消す」を押して、消したい範囲をなぞる
- 全消し: 確認なしで即実行
- 二本指ダブルタップ: 戻る
- 三本指ダブルタップ: 進む
